This repo contains Granby Consulting's builds of GCC 6.20

The supported targets are:

	arm-eabi              Bare Metal
	arm-linux-gnueabi     Linux
	arm-linux-gnueabihf   Linux
	i686-pc-linux         Linux
	m68k-elf              Bare Metal

The tools can be hosted on 64-bit Linux for X86, or on Windows using MINGW32.
